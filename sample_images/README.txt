Sample Images for Quick Image Presenter

This folder demonstrates the image naming convention used by the Quick Image Presenter app.

Naming Convention:
- Format: filename-number.jpg
- Example: slide1-5.jpg means display for 5 seconds
- If no number is specified, default time is used

Examples:
1a-11.jpg     -> displays for 11 seconds
1b-12.jpg     -> displays for 12 seconds  
a1-10.jpg     -> displays for 10 seconds
image.jpg     -> uses default time (5 seconds)

To test the app:
1. Add some images to this folder
2. Use the naming convention above to specify display times
3. Run the Quick Image Presenter app
4. Select this folder when prompted
5. Start the presentation

Supported formats: JPG, PNG, BMP, GIF, TIFF, WebP
